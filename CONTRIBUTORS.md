# Contributors / ပါဝင်ကူညီသူများ

ဒီစာအုပ်ကို issue reports၊ typo fixes၊ wording suggestions၊ diagrams၊ technical corrections နဲ့ discussion တွေကနေ ပိုကောင်းလာအောင် ကူညီပေးသူအားလုံးကို ကျေးဇူးတင်ပါတယ်။

ဒီစာမျက်နှာက contribution acknowledgement အတွက်ဖြစ်ပြီး manuscript ownership သို့မဟုတ် author voice ကို မပြောင်းလဲပါဘူး။

## Maintainer / စာရေးသူနှင့် Maintainer

| Contributor | Role |
|---|---|
| [@htooayelwinict](https://github.com/htooayelwinict) | Author and maintainer |

## Community Feedback / Community မှ ကူညီပေးသူများ

လက်ခံအသုံးပြုထားတဲ့ issue report၊ wording improvement၊ technical correction သို့မဟုတ် တခြား contribution ရှိလာရင် contributor ရဲ့ GitHub profile နဲ့ သက်ဆိုင်ရာ issue သို့မဟုတ် pull request ကို ဒီနေရာမှာ ချိတ်ပြီး မှတ်တမ်းတင်ပါမယ်။

## Acknowledgement Rule

Contribution တစ်ခုကို review လုပ်ပြီး စာအုပ် သို့မဟုတ် repository ထဲမှာ လက်ခံအသုံးပြုပြီးမှ ဒီစာရင်းမှာ ထည့်ပါမယ်။ Commit ရှိမှသာ contributor ဖြစ်တယ်လို့ မကန့်သတ်ဘဲ accepted issue reports နဲ့ editorial feedback တွေကိုလည်း contribution အမျိုးအစားအတိုင်း အသိအမှတ်ပြုပါမယ်။
